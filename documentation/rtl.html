RTL Setup

> gulp rtl

gulp rtl command generate an rtl version of sytle.css and bootstarp.css files. Use this command after adding or editing codes to regenerate those file again if necessary.

It's easy to add RTL support to your website. We have already generated rtl version of syle.css and bootstrap.css files. Just link those files in place of these (syle.css and bootstrap.css) files. Do not use both version of code at the same time.
