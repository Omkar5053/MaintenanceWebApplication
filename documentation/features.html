Features

StrikingDash is multipurpose HTML Admin & Dashboard Template built with Bootstrap 4. Clean professional design and a huge number of shortcode elements are the main features of StrikingDash. We've developed it using latest trend-tools and best practices to leverage the development time and complexity also to ease customization process. We've used configurable SCSS for more control over templates style and layout. We structured it such a way that user can easily compose a new page using our component and customize it without too much hassle.
